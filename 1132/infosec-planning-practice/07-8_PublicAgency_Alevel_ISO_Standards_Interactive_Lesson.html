資安短期衝刺班｜互動式解題講義

題組 1 情境

某資通安全責任等級 A 級 之公務機關，主要業務係辦理全國有關全民健康保險之業務活動。
該機關主責資通安全之承辦窗口，需依據業務特性與法規要求，規劃導入合適的 ISO 國際標準，以強化資訊安全與個人資料保護。

第 7 題 單選題

試問該機關主責資通安全有關事項之承辦窗口，所規畫導入與該機關主要業務特性有關之 ISO 國際標準，下列較為適宜？

名師詳解專區

第一步：找關鍵字

拿起你的螢光筆，這題的解題密碼就在題幹裡：

1. 資通安全責任等級 A 級 + 公務機關：
這代表該單位必須擁有最高規格的資安防護，ISO 27001（資訊安全管理制度）是絕對的基礎標配。

2. 全民健康保險 + 主要業務特性：
健保資料包含姓名、身分證號、病歷、用藥紀錄等，這屬於大量的個人資料（Personal Data）及特種個資（敏感性資料）。這暗示我們除了「資安」之外，必須針對「隱私與個資保護」做加強。

第二步：白話翻譯

各位同學，這題用高中生的話來說就是：
「這個政府機關不只要顧好電腦不被駭客入侵（資安），還因為天天在處理全國人民最私密的健保資料，那它除了基本的資安制度外，還應該搭配哪一種專門在管『個資保護』的國際標準才夠安全？」

  • ISO 27001：資訊安全管理制度的國際標準（像房子的地基與圍牆）。
  • ISO 27701：個人資料保護管理制度（像房子裡專門保管私密日記的保險箱規範，建立在 27001 之上）。

第三步：解題思路

這題在考什麼觀念？
是否能依「業務內容（健保/個資）」選出「最適合搭配 ISO 27001 的延伸標準」。

解題邏輯鏈：

  1. 先確認是 A 級機關 → 必備 ISO 27001（四個選項都有，無法刪去）。
  2. 再看業務性質 → 健保業務涉及海量個資。
  3. 選擇補強標準 → 找跟「個資、隱私」有關的標準 → 鎖定 ISO 27701。

生活比喻：

  • ISO 27001 就像大樓的「門禁與警衛」，防止壞人進來偷東西。
  • ISO 27701 就像醫院的「病歷櫃管理規範」，規定誰可以看病歷、病歷怎麼存、怎麼銷毀。
  • 管健保資料的單位，當然兩個都要有！

第四步：選項分析

選項 搭配標準 分析
(A) ISO 27701 正確。隱私資訊管理。專門針對 PIMS（個人資料管理系統），完全符合健保業務需求。
(B) ISO 45000系列 錯誤。這是「職業安全衛生」，管員工會不會受傷、工安意外，跟資料保護無關。
(C) ISO 17025 錯誤。這是「測試與校正實驗室能力」，通常是檢驗單位（如 SGS）在用的。
(D) ISO 50001 錯誤。這是「能源管理」，管怎麼省電、節能減碳，跟資安無關。

觀念補充與延伸

重要對照表（必背）：

  • ISO 27001 → 資安 (Security)
  • ISO 27701 → 個資/隱私 (Privacy) - 本題核心
  • ISO 22301 → 營運持續 (BCM) - 若題目是電力公司或銀行核心系統常考這個
  • ISO 45001 → 職安 (Safety)
  • ISO 50001 → 能源 (Energy)

記憶口訣

「資安二七零零一，健保個資七七一」
（記法：771 聽起來像「去記憶」，隱私就是要幫你保護記憶資料）

常見陷阱

避坑指南：看到一堆 ISO 編號不要慌。先問自己：這個 ISO 是不是在管「資訊」或「個人資料」？如果是管電費（能源）、管工安意外的，通常在資安題目裡都是誘答選項。

正確答案

答案：(A)

理由：健保業務同時重視資訊安全與個人資料保護，最適合導入 ISO 27001 搭配 ISO 27701。

自我檢測

遮住答案，測測看你學會了嗎？

Q1：ISO 27701 是建立在 ISO 27001 的哪一個基礎之上？
參考答案：資訊安全管理制度（ISMS）
Q2：若組織幾乎不處理個人資料，還一定要導入 ISO 27701 嗎？
參考答案：不一定（視業務需求而定，沒個資就不強求）。
Q3：ISO 45000 主要在管理哪一類風險？
參考答案：職業安全與健康風險。
