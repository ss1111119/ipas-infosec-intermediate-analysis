資安短期衝刺班：觀念特訓

講師：資安補教名師｜單元：風險評鑑流程

28. 一般常見風險評鑑流程包括：「識別背景、風險辨識、風險評估與分析、風險回應、風險監督」。
下列敘述哪些屬於「風險評估與分析」階段？（複選）

【教學解析】

正確答案：C、D

【第一步：找關鍵字】

請特別注意題幹中的：風險評鑑流程、風險評估與分析。

選項中的關鍵字：區分高、中、低風險、定量方法、排序。

說明：本題的關鍵在於判斷這個動作是在「分析風險」，還是在「做決策」。

【第二步：白話翻譯】

「各位同學，試著把公司的風險管理想成這樣：
有些步驟是在『算風險有多大』，有些步驟是在『決定要不要處理』。
題目就是在問你：下面哪些動作，還停留在『算與分』的階段？」

【第三步：解題思路】

一句話核心考點：

「風險評估與分析 ＝ 量化、分類、排序，但還沒做決策」

【原理與規則：建立完整流程觀】

一般風險評鑑流程可簡化理解為：

  1. 風險辨識：找出「有哪些風險」（列清單）。
  2. 風險評估與分析：分析「哪些風險比較大、比較小」，進行分類、排序、量化。
  3. 風險回應：決定「要不要處理、怎麼處理」。

【第四步：選項分析】

  • (A) 執行風險評鑑作業發現 68 個風險項目
    → 這是 風險辨識，只是列出清單，尚未分析大小。
  • (B) 決定維持現狀不做改變
    → 關鍵字「決定」、「維持現狀」，表示已經進入 風險回應（接受風險）。
  • (C) 將風險項目區分為高、中、低 (正確)
    → 這是典型的 風險評估與分析。
  • (D) 使用定量的方法，將風險進行排序 (正確)
    → 明確屬於 分析風險嚴重程度。

【觀念補充與延伸】

快速對照表（考前必記）：

  • 找出有幾個風險 → 風險辨識
  • 分高、中、低／排序 → 風險評估與分析
  • 決定不處理／轉移／降低 → 風險回應
口訣：「先算再決定，算的是分析，決定才是回應」

【常見陷阱與避坑指南】

  • 學生最容易錯：看到「可接受」就以為還在分析，其實「確認可接受」往往是為了做決策的前奏或結果。
  • 避坑指南：題目只要出現「決定」、「維持現狀」，一定已經不是分析階段。

【自我檢測】

  1. 「決定風險是否可接受」屬於哪一階段？（答：風險回應）
  2. 「將風險依數值大小排序」屬於哪一階段？（答：風險評估與分析）
