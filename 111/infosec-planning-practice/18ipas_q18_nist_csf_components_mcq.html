風險管理 網路安全框架（Cybersecurity Framework, CSF）為美國國家標準暨技術研究院（NIST）彙整後所提出，作為整體網路安全架構之規劃藍圖參考，請問該 CSF 之組成元素包含下列哪些項目？（複選）

點擊查看詳細解析
第一步：找關鍵字

請圈出題目中的關鍵字來定位考點：

  • NIST / CSF：鎖定考點為 NIST 的網路安全框架。
  • 組成元素 (Components)：題目問的是框架的「構成部件」，而非內部的具體功能（Functions）。
  • Core / Profile / Tiers：腦中搜尋 CSF 的三大支柱。
第二步：白話翻譯

這題在問：NIST CSF 這套「資安大藍圖」本身是由哪幾個主要零件組成的？

你要選出 CSF 的「三大構成」：核心（Core）、設定檔（Profile）、實施層級（Tiers），並排除掉聽起來很像但其實不是官方元件的字眼（如 Procedure）。

第三步：解題思路

1. 解題邏輯與規則：

  • 框架核心 (Core)：告訴你要做什麼（What to do）。包含 Identify, Protect, Detect, Respond, Recover 等功能。
  • 框架設定檔 (Profile)：告訴你現在在哪、要去哪裡（Current vs Target）。用來找出差距。
  • 實施層級 (Tiers)：告訴你做得多成熟（Maturity）。描述風險管理的嚴謹度。

2. 生活例子比喻：

  • Core (核心) 就像「學校課綱」：規定了大家都要學國英數（標準活動）。
  • Profile (設定檔) 就像「你的成績單 vs 目標分數」：對照你現在考幾分，目標要考幾分，找出要加強哪一科。
  • Tiers (層級) 就像「讀書方法的成熟度」：你是臨時抱佛腳 (Tier 1)，還是有排讀書計畫 (Tier 3)？
第四步：選項分析
  • (A) 框架核心（Framework Core）：正確。這是 CSF 的骨架與共通語言。
  • (B) 框架設定檔（Framework Profile）：正確。這是連結標準與業務需求的橋樑。
  • (C) 實施程序（Framework Procedure）：錯誤。「Procedure」通常指內部的操作流程，並非 NIST CSF 定義的頂層三大元素名稱。
  • (D) 實施層級（Implementation Tiers）：正確。這是衡量風險管理視角與成熟度的工具。
觀念補充與延伸
  • CSF 2.0 更新：雖然組成元素仍維持這三大類，但 CSF 2.0 在 Core 中新增了 "Govern" (治理) 功能，變成 6 大功能。
  • Tiers 的誤區：Implementation Tiers 不是「成熟度模型 (Maturity Model)」，NIST 強調它是用來描述風險管理實務的特性，雖然概念很像，但官方用語較為謹慎。
觸類旁通（舉一反三）

考試若改問「CSF Core 的結構層次」，答案則會是：

  • Function (功能) → Category (類別) → Subcategory (子類別)

考試若問「用來溝通資安現況與目標差距的是哪個元素？」，答案則是：

  • Framework Profile (設定檔)
記憶口訣

「核、檔、層」

  • 核：Core（核心）
  • 檔：Profile（設定檔）
  • 層：Tiers（實施層級）
常見陷阱與避坑指南

心理盲點：

  • 考生常把「CSF 三大元素 (Elements)」和「Core 五大/六大功能 (Functions)」搞混。看到 Identify, Protect 會想選，但題目問的是「組成元素」。
  • 看到「Procedure (程序)」這類專業名詞覺得很合理就選了，但它不是 CSF 的專有名詞。

避坑指南：

  • 看到「組成元素」或「Components」，先默念「Core, Profile, Tiers」這三個單字，找不到的一律刪除。

自我檢測

點擊題目即可查看答案

Q1. CSF Core 內部結構通常分為哪三層（由大到小）？

Function（功能）→ Category（類別）→ Subcategory（子類別）。

Q2. 用來描述組織風險管理「成熟度」與「嚴謹度」（Tier 1 ~ Tier 4）的是哪個元素？

實施層級（Implementation Tiers）。

Q3. 組織想要比較「Current Profile」與「Target Profile」是為了找出什麼？

差距（Gaps），進而制定改善計畫。

Q4. 「實施程序 (Procedure)」是 NIST CSF 的三大組成元素之一嗎？

不是。CSF 三大元素只有 Core、Profile、Tiers。
